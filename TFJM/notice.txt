Pour lancer le serveur :

Pour linux :
./manage.py runserver (manage.py est exécutable)

Pour windows :
ouvrir l'invite de commande.
rendez vous dans le dossier contenant le fichier manage.py grâce à la commande cd
pour lancer le serveur : 
python manage.py runserver
(Si ce n'est pas déjà fait il faut bien sûr installer python)

##################################################################################

normalement l'application se lance dans le terminal.
Pour consulter le site depuis un navigateur web,
taper dans la barre d'adresse :

localhost:8000

ce qui charge la page d'accueil

Pour ouvrir l'outil d'édition :

localhost:8000/admin

et se connecter avec le login/mdp fournis.
(en profiter pour modifier le mot de passe)
